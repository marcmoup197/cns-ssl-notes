# cns-ssl-notes

## 简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容主要围绕 SSL/TLS 相关技术笔记、参考资料或实验记录，以静态 HTML 形式存储，方便通过 GitHub Pages 或其他静态托管服务直接访问。

仓库不针对任何特定域名或具体网站，仅作为个人技术笔记的整理与备份。

## 目录说明

```
cns-ssl-notes/
├── index.html          # 入口页面，用于导航或说明
├── pages/              # 存放各独立 HTML 页面
│   ├── note-1.html
│   ├── note-2.html
│   └── ...
├── assets/             # 静态资源（可选）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- **index.html**：仓库的默认入口页面，建议作为目录或简单介绍。
- **pages/**：所有独立 HTML 页面均存放于此，每个文件为一个独立的笔记或文档。
- **assets/**：存放 CSS、JavaScript、图片等公共资源，便于页面引用。

## 页面归档说明

- 所有页面均为纯前端静态 HTML，不依赖后端服务。
- 页面间可能存在超链接跳转，但每个页面均可独立打开。
- 内容为个人 SSL/TLS 相关学习笔记、配置示例、问题排查记录等，仅供参考。
- 不保证内容的完整性、准确性或时效性。

## 维护说明

- 本仓库不定期更新，新增页面或修改已有内容。
- 欢迎通过 Issue 或 Pull Request 提出建议或修正。
- 更新时请保持页面格式统一，并尽量在 `pages/` 下按逻辑命名文件。

## 使用方式

1. 克隆或下载本仓库。
2. 直接打开 `index.html` 或 `pages/` 下的任意 `.html` 文件查看内容。
3. 若启用 GitHub Pages，可在仓库设置中开启，通过 `https://<用户名>.github.io/cns-ssl-notes/` 访问。

## 许可

本项目内容仅供学习参考，不构成任何形式的保证或承诺。如需引用，请注明来源。
